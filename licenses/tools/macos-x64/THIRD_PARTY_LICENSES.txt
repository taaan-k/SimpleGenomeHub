License Compatibility Guide
--------------------------

**Commercial Use Friendly:**
- BLAST+ (Public Domain)
- MUSCLE (Public Domain)
- MEME Suite (BSD 3-Clause)
- HMMER (BSD 3-Clause)
- ViennaRNA Package (Custom, allows commercial use)

**GPL Copyleft Awareness Required:**
- DIAMOND (GPL v3)
- IQ-TREE (GPL v2)
- Cygwin Runtime Library (GPL)

**Academic Citation Requirements:**
- MCScanX (Mandatory citation)
- ViennaRNA Package (Recommended citation)
- All tools (Academic integrity requires proper citation)
