TBtools User Notice
===================

IMPORTANT: Please read this notice file carefully to understand your rights and obligations when using the TBtools software package.

Software Package Contents
--------------------------

TBtools is a software platform that integrates multiple bioinformatics tools, including the following main components:

1. **Core Software**: TBtools main program
2. **Third-Party Tools**: 32 bioinformatics analysis tools
3. **Runtime Libraries**: Cygwin compatibility layer and related library files

License Compliance Requirements
-------------------------------

When using the TBtools software package, you must comply with the following license requirements:

**Academic Use:**
- All tools can be used free of charge for academic research
- When publishing papers, you must properly cite relevant tools (see citation list)
- It is recommended to mention the tool versions used in the methods section of papers

**Commercial Use:**
- BLAST+ and MUSCLE can be freely used for commercial purposes
- MEME Suite and ViennaRNA Package allow commercial use
- GPL licensed tools (DIAMOND, IQ-TREE) require special attention
- Commercial use of MCScanX requires permission from the original author

**Distribution Requirements:**
- When distributing TBtools, you must include all license files
- Do not delete or modify any copyright notices
- Distribution of GPL components must follow GPL copyleft requirements

Special Considerations
---------------------

1. **GPL Copyleft Effects:**
   - DIAMOND, IQ-TREE use GPL licenses
   - Derivative works deeply integrated with these tools may need to be released under GPL
   - Cygwin runtime libraries may also affect license compatibility

2. **Citation Requirements:**
   - MCScanX: Must cite Wang et al. (2012)
   - ViennaRNA Package: Recommended to cite Lorenz et al. (2011)
   - Other tools: Academic integrity requires proper citation

3. **Version Management:**
   - This license documentation is based on tool versions from February 2026
   - Tool updates may change license terms
   - It is recommended to regularly check for license updates

Quick Citation Format
---------------------

When citing tools used by TBtools in papers, the following format is recommended:

"Analysis used TBtools software [citation], with sequence alignment using BLAST+ [Camacho et al., 2009], 
phylogenetic analysis using IQ-TREE [Nguyen et al., 2015], and gene collinearity analysis using MCScanX [Wang et al., 2012]."

For the complete citation list, see: licenses/THIRD_PARTY_LICENSES.txt
